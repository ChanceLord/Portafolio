¿Qué Tipo de Web Necesito?

Landing Page

Ejemplo


Estructura

Esta es una página única en la que se coloca todo el contenido sin tener que navegar hacia otras páginas. No es una web completa, sino una página informativa, con el fin de lograr un objetivo concreto (suscripción, venta, etc)

Contenido

Se coloca un relato o explicación sobre aquello que se desea vender o promocionar, Debemos desarrollar el texto de forma clara, esquematizada y concisa. Definitivamente en esta página no debe faltar una llamada a la acción. Y casi siempre tiene un sistema de pago o suscripción.

Objetivo

Se utiliza para dar información sobre un producto o servicio concreto. Generalmente, el usuario llega allí a través de un link con la intención de que se realice una única acción concreta, como comprar un producto, contratar un servicio o suscribirte a un newsletter. Se usa generalmente para el lanzamiento de un producto o servicio.




One Page

Ejemplo


Estructura

Este tipo son páginas web completas, que solo constan de una página. Todo su contenido normalmente se coloca en varias páginas a través de las cuales hay que navegar, en una sola página se coloca todo en ella, utilizando un sistema de secciones. Cada sección trata de un tema relacionado con la marca.

Contenido

Es el mismo que colocamos en la página web de varias páginas, pero de manera mucho más sintética. Cada sección de la página tendrá; sobre nosotros, portafolio de productos o servicios, características, contacto, etc. Solo que sería de forma sintética, lineal y consecutiva

Objetivo

Se utilizan generalmente con fines informativos. El uso es similar al de una web tradicional, pero para marcas que están comenzando o necesitan validar su idea o no necesitan extenderse demasiado contando lo que hacen. Es usada comúnmente como portafolio de creativos a los que solo les interesa mostrar sus trabajos. Son ideales para tener presencia online e invertir una menor cantidad de dinero.


Un ejemplo de este tipo de página es esta misma en donde te encuentras




Página Web Informativa

Ejemplo


Estructura

es similar a la one page solo que con diferentes secciones que suceden de forma lineal con contenido variado sobre la marca. Es una estructura tradicional, aunque puede ser de muchos tipos, por ejemplo, secciones que funcionan a pantalla completa con scroll horizontal o que sea únicamente una pantalla sin scroll.

Contenido

En una web o varias páginas, debemos colocar un contenido fuerte, bien desarrollado y llamativo. En la página de inicio, debe haber textos o elementos que invitan a navegar a través de sus páginas. Por lo que, el contenido de la página de inicio, debe ser sintético, claro y directo. Lo más importante debe ser llamativo y causar una buena primera impresión.

Objetivo

Como su nombre lo indica es lo primero que tu prospecto verá al entrar tu página web. Por lo que sirve para dar a conocer tu marca, trayectoria, experiencia, productos y servicios, pero de una forma resumida. Para invitar al usuario a navegar a través de las demás páginas de tu web, con llamadas a la acción. Por lo que, su función principal es dar un bosquejo de tu marca, servir de índice para el resto de la web y dar acceso rápido a información práctica.
